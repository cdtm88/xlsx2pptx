# assets

Brand assets for the generator pages.

## logo

`logo.svg` is a placeholder. To use your own logo:

1. In GitHub, open `assets/logo.svg` (or add `assets/logo.png`).
2. Replace it with your file, keeping it in this `assets/` folder.
3. The logo is embedded into the page header as a data URI (so the HTML
   stays self-contained and works offline) — after you upload, it gets
   re-embedded from this file.

SVG is preferred (crisp at any size, tiny). PNG also works.
